# Laravel Portfolio with Admin Panel

A modern, dynamic portfolio website built with Laravel 13, Livewire 4, and Flux UI. Features a fully functional admin panel to manage all portfolio content including profile information, work experiences, projects, skills, and more.

## ✨ Features

### 🎯 Dynamic Portfolio (Frontend)
- Responsive, modern UI with Tailwind CSS
- Fully dynamic content from database
- Animated sections with smooth scrolling
- Skills showcase with proficiency levels
- Project filtering by category
- Timeline display of work experiences
- Contact section with social links
- Expandable insights for each experience

### 🔧 Admin Panel (Backend)
- **Profile Management** - Update personal info, avatar, social links
- **Experience Management** - CRUD for work history
- **Project Management** - Add/edit/delete projects with technologies
- **Skills Management** - Manage technology stack with proficiency levels
- **Technology Management** - Manage technologies with icons and colors
- **Authentication** - Secure login/logout with Laravel Fortify

## 📋 Prerequisites

- PHP >= 8.2
- Composer
- Node.js & NPM
- MySQL (or any supported database)
- Laravel 13 requirements
